# AutoPartHelper RU – AI Assistant & Telegram Bot

## 📖 Описание
Интеллектуальный ИИ-ассистент для комплексного подбора оригинальных и аналоговых автозапчастей, организации технического обслуживания и ремонта легковых автомобилей на территории Российской Федерации. Работает через Telegram-бота, интегрирован с платформой Suvvy.ai и Google Calendar. Предназначен для автовладельцев, частных мастеров и небольших СТО, обеспечивая точный подбор, прозрачное ценообразование, учёт региональной логистики и удобную запись на сервис.

## ⚙️ Задачи и функции
- **Идентификация автомобиля:** Расшифровка VIN-кода (17 символов) или поиск по марке/модели/году выпуска с учётом модификаций двигателя, типа КПП и комплектации.
- **Подбор запасных частей:** Формирование перечня деталей с верифицированными OEM-номерами и проверенными аналогами (Bosch, Mann, Brembo, Textar, TRW и др.) на основе профессиональных каталогов (TecDoc, EPC производителей, открытые базы совместимости).
- **Мониторинг рынка:** Поиск актуальных цен, наличия и условий доставки в интернет-магазинах РФ (Exist.ru, Emex.ru, Autodoc.ru, Zzap.ru, Parts.ru).
- **Организация ТО и ремонта:** Расчёт необходимого комплекта расходников для конкретных операций (замена масла, тормозных колодок, фильтров, ГРМ, подвески и др.), рекомендация проверенных СТО по региону клиента.
- **Запись на сервис:** Интеграция с Google Calendar для бронирования слотов диагностики, консультации или установки запчастей с автоматическим учётом часового пояса и доступных окон.
- **Клиентская поддержка:** Формирование итоговых чек-листов, адаптация стиля общения под уровень знаний клиента (технические спецификации или простые пояснения), проактивные уточнения без разрыва диалога, обработка запросов в произвольном порядке.

## 🛠 Техническая архитектура
- **Оркестрация диалогов:** Suvvy.ai (бесплатный тариф)
- **Языковая модель:** Qwen (Alibaba Cloud) – анализ запросов, генерация ответов, наполнение базы знаний
- **Канал связи:** Telegram Bot API
- **Планировщик:** Google Calendar API (управление слотами, синхронизация времени, уведомления)
- **Источники данных:** Публичные каталоги автопроизводителей, TecDoc, агрегаторы запчастей РФ, открытые базы кросс-номеров

## ✅ Преимущества
- **Точность подбора:** Верификация по VIN и кросс-проверка part numbers в нескольких каталогах минимизирует риск несовместимости и возвратов.
- **Прозрачность:** Фиксация ориентировочных цен на момент запроса, указание источников данных, сроков доставки и условий гарантии, отсутствие скрытых комиссий.
- **Клиентоориентированность:** Гибкий диалог (не требует строгого порядка ввода данных), поддержка разных форматов запросов, персонализированные рекомендации, устойчивость к неполным данным.
- **Региональная адаптация:** Учёт логистики по РФ, поддержка оплаты через СБП/МИР/безнал, приоритет продавцам с российской гарантией и сертификатами ЕАЭС.
- **Бюджетная реализация:** Архитектура построена на бесплатных/открытых инструментах, что позволяет запускать и масштабировать решение без высоких стартовых затрат на лицензионное ПО.

## 📌 Ограничения и рекомендации
- Подбор носит рекомендательный характер; перед оплатой рекомендуется финальная сверка по VIN у поставщика.
- Цены и наличие являются ориентировочными и могут изменяться в зависимости от региона, курса валют и складской политики магазинов.
- Интеграция с платными API (TecDoc, Exist, Emex) может быть добавлена для автоматизации мониторинга в реальном времени.
- Проект не является официальным дилером или автосервисом; все работы и гарантия на детали регулируются третьими лицами.

## 🚀 Быстрый старт
1. Добавьте бота в Telegram и отправьте `/start`.
2. Укажите марку/модель автомобиля или VIN-код, опишите требуемую операцию.
3. Получите чек-лист запчастей с part numbers, сравнение цен и варианты доставки.
4. При необходимости запишитесь на диагностику или установку через встроенный календарь.
